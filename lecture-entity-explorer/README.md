# Lecture Entity Explorer

This folder is the GitHub-ready version of the lecture explorer.

Files:
- `index.html`
- `lecture_entity_explorer_app.js`
- `lecture_entity_data.js`
- `lecture_entity_data.json`

To publish on GitHub Pages:
1. Create a new GitHub repository.
2. Upload all files from this folder to the repository root.
3. In GitHub, enable Pages from the `main` branch root.
4. Open the Pages URL after deployment finishes.

`index.html` is the entry page.
